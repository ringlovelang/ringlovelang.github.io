The Ringlove Programming Language

Ringlove is a kind of object oriented language for people that want a customizable programming language (by using the module keyword registration thing) and others.

class {Test}
  func set (ID=(stuff))
    print{"15", ID(Value)}

  if(stuff.Value == "15") {
    print{"The value is indeed", FromID=(stuff.Value)}
  }
}
